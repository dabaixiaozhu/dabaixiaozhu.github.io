# 基础的三个排序

## 选择排序

### 思路

从“待排序部分”找到最小值（或者最天值），然后交换元素，让“待排序部分”的起始位置向后移动

![selectSort](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class1/selectSort.png)

### 代码

```java
public static void selectionSort(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    // 0～n-1 1～n-1 2～n-1
    // 遍历
    for (int i = 0; i < arr.length - 1; i++) {
        // 最小值在哪个位置上  i～n-1
        int minIndex = i;
        // i ~ N-1 上找最小值的下标
        for (int j = i + 1; j < arr.length; j++) {
            minIndex = arr[j] < arr[minIndex] ? j : minIndex;
        }
        // 循环到的第i个位置跟 i之后的最小值进行位置交换
        swap(arr, i, minIndex);
    }
}

/**
 * 交换输入元素
 */
public static void swap(int[] arr, int i, int j) {
    int tmp = arr[i];
    arr[i] = arr[j];
    arr[j] = tmp;
}
```



## 冒泡排序

### 思路

相邻的元素两两比较，当一个元素大于右侧相邻元素时，交换它们的位置；当一个元素小于或等于右侧相邻元素时，位置不变；元素9为最大的元素，“漂”到了最右侧

![bublesort1](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class1/bubleSort1.png)

每一轮的逻辑如上

![bubleSort2](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class1/bubleSort2.png)

### 代码

```java
public static void bubbleSort(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    // 0 ~ e
    // 最后一位数字为范围内最大值
    for (int e = arr.length - 1; e > 0; e--) {
        for (int i = 0; i < e; i++) {
            // 左值>右值
            if (arr[i] > arr[i + 1]) {
                swap(arr, i, i + 1);
            }
        }
    }
}

// 交换arr的i和j位置上的值
public static void swap(int[] arr, int i, int j) {
    // 异或运算
    arr[i] = arr[i] ^ arr[j];
    arr[j] = arr[i] ^ arr[j];
    arr[i] = arr[i] ^ arr[j];
}
```



## 插入排序

### 思路

不断地把数据插入，**已排序的部分数据列**中的恰当位置

![insertSort](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class1/insertSort.png)

### 代码

```java
public static void insertionSort(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    // 0~0 有序的 0~i 想有序
    // 0 ~ i 做到有序
    for (int i = 1; i < arr.length; i++) {
        // arr[i]往前看，一直交换到合适的位置停止
        for (int j = i - 1; j >= 0 && arr[j] > arr[j + 1]; j--) {
            // 只要后面的值更小，就往前面插入
            swap(arr, j, j + 1);
        }
    }
}

// i和j是一个位置的话，会出错
public static void swap(int[] arr, int i, int j) {
    arr[i] = arr[i] ^ arr[j];
    arr[j] = arr[i] ^ arr[j];
    arr[i] = arr[i] ^ arr[j];
}
```



# 对数器

## 思路

- 1 你想要测的方法a
- 2 实现复杂度不好但是容易实现的方法b
- 3 实现一个随机样本产生器
- 4 把方法a和方法b跑相同的随机样本，看看得到的结果是否一样
- 5 如果有一个随机样本使得比对结果不一致，打印样本进行人工干预，改对方法a和方法b
- 6 当样本数量很多时比对测试依然正确，可以确定方法a已经正确

## 代码

### 得到随机长度、随机内容的数组

```java
public static int[] generateRandomArray(int maxSize, int maxValue) {
    // Math.random() [0,1) 任意小数
    // Math.random()*N [0,N) 任意小数
    // (int)(Math.random()*N)  [0,N-1] 任意整数
    int[] arr = new int[(int) ((maxSize + 1) * Math.random())]; // 随机长度的数组
    for (int i = 0; i < arr.length; i++) {
        // 随机整数 - 随机整数
        arr[i] = (int) ((maxValue + 1) * Math.random()) - (int) (maxValue * Math.random());
    }
    return arr;
}
```

# 评估算法优劣的核心指标

## 重要性递减

- 时间复杂度：经历了多少次常数时间的操作

- 额外空间复杂度

- 常数项时间：直接大样本去比较

## 常数时间的操作

执行时间固定的操作都是常数时间的操作，执行时间不固定的操作，都不是常数时间的操作

- 常见的算术运算（+、-、*、/、% 等）
- 常见的位运算（>>、>>>、<<、|、&、^等）
  - \>>：带符号位右移，最高位整数用0补，负数用1补
  - \>>>：不带符号位右移，最高位用0补位
- 赋值、比较、自增、自减操作等
- 数组寻址操作：a[1]，直接算偏移量

## 1 在一个有序数组中，找某个数是否存在

### 思路

![BS1](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class1/BS1.png)

### 代码

```java
public static boolean exist(int[] sortedArr, int num) {
    if (sortedArr == null || sortedArr.length == 0) {
        return false;
    }
    int L = 0;
    int R = sortedArr.length - 1;
    int mid = 0;
    // L<R 才能取中间值
    while (L < R) {
        // 防止数值过大溢出
        mid = L + ((R - L) >> 1); // mid = (L + R) / 2
        if (sortedArr[mid] == num) {
            return true;
        }
        // 右侧内容
        else if (sortedArr[mid] > num) {
            R = mid - 1;
        }
        // 左侧内容
        else {
            L = mid + 1;
        }
    }
    return sortedArr[L] == num;
}
```



## 2 在一个有序数组中，找>=目标值最左侧的位置

### 思路

![BS2](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class1/BS2.png)

### 代码

```java
public static int nearestIndex(int[] arr, int value) {
    int L = 0;
    int R = arr.length - 1;
    int index = -1; // 记录最左的对号
    while (L <= R) {
        // 取中间值
        int mid = L + ((R - L) >> 1);
        // 中间值大于目标值，意味着目标值在[左侧,中间值]
        if (arr[mid] >= value) {
            index = mid;
            // 新的右侧
            R = mid - 1;
        } else {
            // 新的左侧
            L = mid + 1;
        }
    }
    // 没有符合的情况，返回-1
    return index;
}
```

## 4 局部最小值问题

### 思路

![BS4](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class1/BS4.png)

### 代码

```java
public static int getLessIndex(int[] arr) {
    if (arr == null || arr.length == 0) {
        return -1; // no exist
    }
    // 首位局部最小值
    if (arr.length == 1 || arr[0] < arr[1]) {
        return 0;
    }
    // 末尾局部最小值
    if (arr[arr.length - 1] < arr[arr.length - 2]) {
        return arr.length - 1;
    }
    int left = 1;
    int right = arr.length - 2;
    int mid = 0;
    while (left < right) {
        mid = (left + right) / 2;
        // 大于任意一边 则不满足条件
        // 中间值大于哪一边 就往哪一遍缩小范围 默认是左侧
        if (arr[mid] > arr[mid - 1]) {
            right = mid - 1;
        } else if (arr[mid] > arr[mid + 1]) {
            left = mid + 1;
        } else {
            return mid;
        }
    }
    return left;
}
```

```java
public static void printOddTimesNum2(int[] arr) {
    int eor = 0;
    for (int i = 0; i < arr.length; i++) {
        eor ^= arr[i];
    }
    // eor = a ^ b
    // eor != 0
    // eor必然有一个位置上是1
    int rightOne = eor & (~eor + 1); // 提取出最右的1 题目四
    int onlyOne = 0; // eor'
    for (int i = 0 ; i < arr.length;i++) {
        // 比如 第八位 数字为1 那一类
        if ((arr[i] & rightOne) != 0) {
            onlyOne ^= arr[i];
        }
    }
    // eor ^ onlyOne 就代表另一个结果
    System.out.println(onlyOne + " " + (eor ^ onlyOne));
}
```

7 获取二进制中1的个数

```java
public static int bitcount(int N){
    int count =0;

    while (N!=0){
        // 提取最右侧的1
        int rightOne = N & ((~N)+1);
        // 抹掉最右侧的1 让rightOne 与 N 相同的部分为0，不同的为1
        N ^= rightOne;
        count++;
    }
    return count;
}
```
