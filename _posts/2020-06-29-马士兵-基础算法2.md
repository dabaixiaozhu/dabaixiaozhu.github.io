```java
public static Node reverseLinkedList(Node head) {
    Node pre = null;
    Node next = null;
    while (head != null) {
        // next只是记一下环境
        next = head.next;
        // 改变指针方向
        head.next = pre;
        pre = head;
        head = next;
    }

}
```

#### 思路：while3次

![deleteFormNum](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class2/deleteFormNum.png)

#### 代码

```java
public static Node removeValue(Node head, int num) {
    // dead来到了第一个不需要删除的位置
    while (head != null) {
        // 头部不需要删除
        if (head.value != num) {
            break;
        }
        // 头部需要删除的情况，头部指向下一个值
        head = head.next;
    }
    // 到这后 head不要动了
    // 此时的pre肯定是不需要删除的，即第一个不需要删除的Node
    // 设置不需要删除的Node 定义为pre
    Node pre = head;
    // 需要删除的Node
    Node cur = head;
    while (cur != null) {
        // 代表cur该删了
        if (cur.value == num) {
            pre.next = cur.next;
        } else {
            pre = cur;
        }
        cur = cur.next;
    }
    return head;
}
```

### 代码

#### 双向链表实现头部进/出  尾部进/出

```java
public static class DoubleEndsQueue<T> {
    // 头节点
    public Node<T> head;
    // 尾节点
    public Node<T> tail;

    // 从头部进
    public void addFromHead(T value) {
        // 要插入的节点
        Node<T> cur = new Node<T>(value);
        // 链表中一个值都没有
        if (head == null) {
            // 头尾都是添加的value
            head = cur;
            tail = cur;
        }
        // 头部不为空
        else {
            // 见笔记
            cur.next = head;
            head.last = cur;
            head = cur;
        }
    }

    // 从头部弹出节点
    public T popFromHead() {
        // 链表里啥都没有 返回为空
        if (head == null) {
            return null;
        }
        // 如果链表里有内容
        Node<T> cur = head;
        // 链表中只有一个元素，链表为空
        if (head == tail) {
            head = null;
            tail = null;
        }
        // 链表中不止一个元素
        else {
            head = head.next;
            cur.next = null;
            head.last = null;
        }
        return cur.value;
    }

    // 从尾部加节点
    public void addFromBottom(T value) {
        Node<T> cur = new Node<T>(value);
        // 链表中一个值都没有
        if (head == null) {
            // 头尾都是添加的value
            head = cur;
            tail = cur;
        }
        // 链表不为空
        else {
            cur.last = tail;
            tail.next = cur;
            tail = cur;
        }
    }

    // 从尾部弹出内容
    public T popFromBottom() {
        if (head == null) {
            return null;
        }
        Node<T> cur = tail;
        if (head == tail) {
            head = null;
            tail = null;
        } else {
            tail = tail.last;
            tail.next = null;
            cur.last = null;
        }
        return cur.value;
    }

    // 判断是否为空
    public boolean isEmpty() {
        return head == null;
    }
}
```

#### 双向链表实现栈

```java
public static class MyStack<T> {
    // 用双向链表实现
    private DoubleEndsQueue<T> queue;

    // 创建栈
    public MyStack() {
        queue = new DoubleEndsQueue<T>();
    }

    // 入栈
    public void push(T value) {
        queue.addFromHead(value);
    }

    // 出栈
    public T pop() {
        return queue.popFromHead();
    }

    // 栈中是否为空
    public boolean isEmpty() {
        return queue.isEmpty();
    }
}
```

## 数组实现（环形数组）

### 思路

![RingArray](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class2/RingArray.png)

### 代码

```java
public static class MyQueue {
    private int[] arr;
    private int pushi;
    private int polli;
    private int size;
    private final int limit;

    // 构造方法
    public MyQueue(int l) {
        // 固定数组长度
        arr = new int[l];
        // 插入位置
        pushi = 0;
        // 弹出位置
        polli = 0;
        // 数组包含的元素个数
        size = 0;
        // 数组限制长度
        limit = l;
    }

    // 如果现在下标是i 返回下一个位置 弹出/插入情况相同
    // 极限情况 i=limit-1 即最后一个位置 插入位置变为0
    private int nextIndex(int i) {
        return i < limit - 1 ? i + 1 : 0;
    }

    // 往数组中添加元素
    public void push(int value) {
        // 数组元素已达最大值
        if (size == limit) {
            throw new RuntimeException("栈满了，不能再加了");
        }
        size++;
        // 插入位置赋值
        arr[pushi] = value;
        // 插入位置 的下一个位置
        pushi = nextIndex(pushi);
    }

    // 往数组中弹出元素
    public int pop() {
        if (size == 0) {
            throw new RuntimeException("栈空了，不能再拿了");
        }
        size--;
        // 要弹出的元素
        int ans = arr[polli];
        polli = nextIndex(polli);
        return ans;
    }
}
```

## 实现一个特殊的栈，在基本功能的基础上，再实现返回栈中最小元素的功能

### 要求

- 1 pop、push、getMin操作的时间复杂度都是 O(1)
- 2 设计的栈类型可以使用现成的栈结构

### 思路

![specialStack](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class2/specialStack.png)

### 代码

```java
public static class MyStack2 {
    // 普通栈
    private Stack<Integer> stackData;
    // 最小栈
    private Stack<Integer> stackMin;

    public MyStack2() {
        this.stackData = new Stack<Integer>();
        this.stackMin = new Stack<Integer>();
    }

    // 添加栈元素
    public void push(int newNum) {
        // 最小栈插入元素
        // 最小栈为空 则插入第一个数
        if (this.stackMin.isEmpty()) {
            this.stackMin.push(newNum);
        }
        // 新元素小于栈顶，插入新元素
        else if (newNum < this.getmin()) {
            this.stackMin.push(newNum);
        }
        // 当前栈栈顶更小，插入栈顶元素
        else {
            int newMin = this.stackMin.peek();
            this.stackMin.push(newMin);
        }
        // 普通栈插入元素
        this.stackData.push(newNum);
    }

    // 弹出正常栈 栈顶元素
    public int pop() {
        if (this.stackData.isEmpty()) {
            throw new RuntimeException("Your stack is empty.");
        }
        return this.stackData.pop();
    }

    // 获取最小栈 栈顶
    public int getmin() {
        if (this.stackMin.isEmpty()) {
            throw new RuntimeException("Your stack is empty.");
        }
        // 弹栈
        return this.stackMin.peek();
    }
}
```

### 思路

### 代码

#### 两个队列实现栈

```java
// 两个队列来实现栈
public static class TwoQueueStack<T> {
    // Data队列
    public Queue<T> queue;
    // help队列
    public Queue<T> help;

    public TwoQueueStack() {
        queue = new LinkedList<>();
        help = new LinkedList<>();
    }

    // 添加元素 Data队列
    public void push(T value) {
        queue.offer(value);
    }

    // 弹出元素
    public T poll() {
        // 只要Data队列里除最后一个元素的其他元素
        while (queue.size() > 1) {
            // help队列里就添加元素
            help.offer(queue.poll());
        }
        // 弹出Data队列剩余的最后一个元素
        T ans = queue.poll();
        // 换名字
        Queue<T> tmp = queue;
        queue = help;
        help = tmp;
        return ans;
    }

    // 获取最后一个元素（不弹出）
    public T peek() {
        while (queue.size() > 1) {
            help.offer(queue.poll());
        }
        T ans = queue.peek();
        Queue<T> tmp = queue;
        queue = help;
        help = tmp;
        help.offer(ans);
        return ans;
    }
}
```

#### 两个栈实现队列

```java
// 两个栈来实现队列
public static class TwoStacksQueue {
    // 两个栈
    public Stack<Integer> stackPush;
    public Stack<Integer> stackPop;

    // 构造函数
    public TwoStacksQueue() {
        stackPush = new Stack<Integer>();
        stackPop = new Stack<Integer>();
    }

    // push栈向pop栈倒入数据
    private void pushToPop() {
        // 遵循两个原则
        // 1 pop栈必须为空
        if (stackPop.empty()) {
            // 2 push栈必须全部导出
            while (!stackPush.empty()) {
                stackPop.push(stackPush.pop());
            }
        }
    }

    // push栈添加数据  看添加元素对于结果是否有影响
    public void add(int pushInt) {
        stackPush.push(pushInt);
        pushToPop();
    }

    // pop栈弹出数据  看弹出元素对于结果是否有影响
    public int poll() {
        if (stackPop.empty() && stackPush.empty()) {
            throw new RuntimeException("Queue is empty!");
        }
        pushToPop();
        return stackPop.pop();
    }

    // 返回栈顶元素 返回pop栈 栈顶元素
    public int peek() {
        if (stackPop.empty() && stackPush.empty()) {
            throw new RuntimeException("Queue is empty!");
        }
        pushToPop();
        return stackPop.peek();
    }
}
```



# 递归

## 用递归方法实现求数组arr[L..R]中的最大值

### 思路

- 1 将[L..R]范围分成左右两半。左：[L..Mid]  右[Mid+1..R]
- 2 左部分求最大值，右部分求最大值（当范围上只有一个数，就可以不用再递归了）
- 3 [L..R]范围上的最大值，是max{左部分最大值，右部分最大值

![getMax](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class2/getMax.png)

### 代码

```java
// 求arr中的最大值
public static int getMax(int[] arr) {
    return process(arr, 0, arr.length - 1);
}

// arr[L..R]范围上求最大值 假设L到R 有n个数
public static int process(int[] arr, int L, int R) {
    // arr[L..R]范围上只有一个数，直接返回，base case
    if (L == R) {
        return arr[L];
    }
    //  L..mid  mid+1...R
    // int mid = (L+R)/2
    int mid = L + ((R - L) >> 1); // 中点
    // 左边范围最大值
    int leftMax = process(arr, L, mid);
    // 右边范围最大值
    int rightMax = process(arr, mid + 1, R);
    // 两个数相比，得到整个数组的最大值
    return Math.max(leftMax, rightMax);
}
```

## Master公式

### 定义

T(N) = a * T(N/b) + O(N^d)(其中的a、b、d都是常数)的递归函数，可以直接通过Master公式来确定时间复杂度

![Master](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class2/Master.png)

### 计算复杂度

- 如果 log(b,a) < d，复杂度为O(N^d)
- 如果 log(b,a) > d，复杂度为O(N^log(b,a))
- 如果 log(b,a) == d，复杂度为O(N^d  * logN)

# 哈希表

## 特点

- 1 哈希表在使用层面上可以理解为一种集合结构
- 2 如果只有key，没有伴随数据value，可以使用HashSet结构
- 3 如果既有key，又有伴随数据value，可以使用HashMap结构
- 4 有无伴随数据，是HashMap和HashSet唯一的区别，实际结构是一回事
- 5 使用哈希表增(put)、删(remove)、改(put)和查(get)的操作，可以认为时间复杂度为 O(1)，但是常数时间比较大
- 6 放入哈希表的东西，如果是基础类型，内部按值传递，内存占用是这个东西的大小
- 7 放入哈希表的东西，如果不是基础类型，内部按引用传递，内存占用是8字节

# 有序表

## 特点

- 1 有序表在使用层面上可以理解为一种集合结构
- 2 如果只有key，没有伴随数据value，可以使用TreeSet结构
- 3 如果既有key，又有伴随数据value，可以使用TreeMap结构
- 4 有无伴随数据，是TreeSet和TreeMap唯一的区别，底层的实际结构是一回事
- 5 有序表把key按照顺序组织起来，而哈希表完全不组织
- 6 红黑树、AVL树、size-balance-tree和跳表等都属于有序表结构，只是底层具体实现不同
- 7 放入如果是基础类型，内部按值传递，内存占用就是这个东西的大小
- 8 放入如果不是基础类型，内部按引用传递，内存占用是8字节
- 9 不管是什么底层具体实现，只要是有序表，都有以下固定的基本功能和固定的时间复杂度
- 10 有序表在使用时，比哈希表功能多，时间复杂度都是O(logN)

## 代码
