# <a name="1">[1 前缀树](#begin)</a>

## <a name="1.1">[概念](#begin)</a>

- 1 单个字符串中，字符从前到后的加到一棵多叉树上
- 2 字符放在路上，节点上有专属的数据项（常见的是pass和end值）
- 3 所有样本都这样添加，如果没有路就新建，如有路就复用
- 4 沿途节点的pass值增加1，每个字符串结束时来到的节点end值增加1

### <a name="1.21">[1 添加某个字符串，可以重复添加，每次算1个](#begin)</a>

```java
// 1 加入一个word
public void insert(String word) {
    if (word == null) {
        return;
    }
    char[] str = word.toCharArray();
    Node1 node = root;
    // 要新加字符串 头节点的pass先++
    node.pass++;
    int path = 0;
    // 遍历传入的字符串中的字符
    for (int i = 0; i < str.length; i++) { // 从左往右遍历字符
        // 由字符，对应成走向哪条路 a即0的位置的路
        path = str[i] - 'a';
        // 当前这条路下一个节点是否为null
        if (node.nexts[path] == null) {
            // 在此方向上建立新节点
            node.nexts[path] = new Node1();
        }
        // 跳到下一个节点
        node = node.nexts[path];
        // 只要到了下个节点 pass++
        node.pass++;
    }
    // 遍历完一个字符串，这里的node是最后一个节点了 end++
    node.end++;
}
```

```java
public void delete(String word) {
    // 必须要有此路径 才能删除
    if (search(word) != 0) {
        char[] chs = word.toCharArray();
        // 头节点
        Node1 node = root;
        // 头节点的p先要-1
        node.pass--;
        int path = 0;
        for (int i = 0; i < chs.length; i++) {
            // 下一级的路
            path = chs[i] - 'a';
            // 下一级节点p值-- 后为0
            if (--node.nexts[path].pass == 0) {
                // 设置此节点为空 直接return下面的内容不要了
                node.nexts[path] = null;
                return;
            }
            // 下一个节点
            node = node.nexts[path];
        }
        node.end--;
    }
}
```

```java
// 实现1
public static class Node1 {
    public int pass;
    public int end;
    public Node1[] nexts;

    public Node1() {
        pass = 0;
        end = 0;
        // 每个节点下一条路 最多就是从26个字母中选一个
        // 0    a
        // 1    b
        // 2    cd
        // ..   ..
        // 25   z
        // nexts[i] == null   i方向的路不存在
        // nexts[i] != null   i方向的路存在
        nexts = new Node1[26];
    }
}

public static class Trie1 {
    private Node1 root;

    // 初始化只需要头节点
    public Trie1() {
        // 初始化的节点 p e默认为0
        root = new Node1();
    }

    // 1 加入一个word
    public void insert(String word) {
        if (word == null) {
            return;
        }
        char[] str = word.toCharArray();
        Node1 node = root;
        // 要新加字符串 头节点的pass先++
        node.pass++;
        int path = 0;
        // 遍历传入的字符串中的字符
        for (int i = 0; i < str.length; i++) { // 从左往右遍历字符

            path = str[i] - 'a';
            // 当前这条路下一个节点是否为null
            if (node.nexts[path] == null) {
                // 在此方向上建立新节点
                node.nexts[path] = new Node1();
            }
            // 跳到下一个节点
            node = node.nexts[path];
            // 只要到了下个节点 pass++
            node.pass++;
        }
        // 遍历完一个字符串，这里的node是最后一个节点了 end++
        node.end++;
    }

    // 2 word这个单词之前加入过几次
    // 这里 p e值不用增加
    public int search(String word) {
        // 不可能加入为空
        if (word == null) {
            return 0;
        }
        char[] chs = word.toCharArray();
        Node1 node = root;
        int index = 0;
        for (int i = 0; i < chs.length; i++) {
            index = chs[i] - 'a';
            // 下标没路了 表示到了最后一个节点

                return 0;
            }
            node = node.nexts[index];
        }
        // 最后一个节点的end值，就是加入过的次数
        return node.end;
    }

    // 3 删除路径
    public void delete(String word) {
        // 必须要有此路径 才能删除
        if (search(word) != 0) {
            char[] chs = word.toCharArray();

            Node1 node = root;
            // 头节点的p先要-1
            node.pass--;
            int path = 0;
            for (int i = 0; i < chs.length; i++) {
                // 下一级的路
                path = chs[i] - 'a';
                // 下一级节点p值-- 后为0
                if (--node.nexts[path].pass == 0) {
                    // 设置此节点为空 直接return下面的内容不要了
                    node.nexts[path] = null;
                    return;

                // 下一个节点
                node = node.nexts[path];
            }
            node.end--;
        }
    }

    // 4 所有加入的字符串中，有几个是以pre这个字符串作为前缀的

            return 0;
```

```java
// 实现2
// 实现1只是字母 如果输入的字符种类很多，就要用哈希表的形式
public static class Node2 {
    public int pass;
    public int end;
    // 哈希表 其余的没有区别
    public HashMap<Integer, Node2> nexts;

    public Node2() {
        pass = 0;
        end = 0;
        nexts = new HashMap<>();
    }
}

public static class Trie2 {
    private Node2 root;

    // 初始化只需要头节点
    public Trie2() {
        root = new Node2();
    }

    // 加入一个word
    public void insert(String word) {
        if (word == null) {
            return;
        }
        char[] chs = word.toCharArray();
        Node2 node = root;
        node.pass++;
        int index = 0;
        for (int i = 0; i < chs.length; i++) {
            // 把字符转为数字
            index = (int) chs[i];
            // 下一个节点是否为空 即map中是否含有此key
            if (!node.nexts.containsKey(index)) {
                // 如果节点为空 则新建节点
                node.nexts.put(index, new Node2());
            }
            // 跳到下一个节点
            node = node.nexts.get(index);
            // 只要到了下个节点 pass++
            node.pass++;
        }
        node.end++;
    }

    // word这个单词之前加入过几次
    // 这里 p e值不用增加
    public int search(String word) {
        if (word == null) {
            return 0;
        }
        char[] chs = word.toCharArray();
        Node2 node = root;
        int index = 0;
        for (int i = 0; i < chs.length; i++) {
            index = (int) chs[i];
            if (!node.nexts.containsKey(index)) {
                return 0;
            }
            node = node.nexts.get(index);
        }
        // 最后一个节点的end值，就是加入过的次数
        return node.end;
    }

    // 删除路径
    public void delete(String word) {
        // 必须要有此路径 才能删除
        if (search(word) != 0) {
            char[] chs = word.toCharArray();
            // 头节点
            Node2 node = root;
            // 头节点的p先要-1
            node.pass--;
            int index = 0;
            for (int i = 0; i < chs.length; i++) {
                index = (int) chs[i];
                // 下一级节点p值-- 后为0
                if (--node.nexts.get(index).pass == 0) {
                    // 删除此节点 直接return下面的内容不要了
                    node.nexts.remove(index);
                    return;
                }
                // 下一个节点的值
                node = node.nexts.get(index);
            }
            node.end--;
        }
    }

    // 所有加入的字符串中，有几个是以pre这个字符串作为前缀的
    public int prefixNumber(String pre) {
        if (pre == null) {
            return 0;
        }
        char[] chs = pre.toCharArray();
        Node2 node = root;
        int index = 0;
        for (int i = 0; i < chs.length; i++) {
            index = (int) chs[i];
            // 不存在此路径
            if (!node.nexts.containsKey(index)) {
                return 0;
            }
            node = node.nexts.get(index);
        }
        // p值则为经过值
        return node.pass;
    }
}
```

# 2 不基于比较的排序

## 计数排序

### 特点

- 计数排序要求，样本是整数，且范围比较窄
- 就是比如到100，就得有100个桶

### 代码

```java
// only for 0~200 value
// 计数排序
public static void countSort(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    // 看要建多少个桶
    int max = Integer.MIN_VALUE;
    for (int i = 0; i < arr.length; i++) {
        max = Math.max(max, arr[i]);
    }
    int[] bucket = new int[max + 1];
    for (int i = 0; i < arr.length; i++) {
        bucket[arr[i]]++;
    }
    // 将桶中的数据再拷贝回arr
    int i = 0;
    for (int j = 0; j < bucket.length; j++) {
        while (bucket[j]-- > 0) {
            arr[i++] = j;
        }
    }
}
```

```java
// only for no-negative value
public static void radixSort(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    // 0,n-1 最大值是几位
    radixSort(arr, 0, arr.length - 1, maxbits(arr));
}

// 获取最大值的位数
public static int maxbits(int[] arr) {
    int max = Integer.MIN_VALUE;
    for (int i = 0; i < arr.length; i++) {
        max = Math.max(max, arr[i]);
    }
    int res = 0;
    while (max != 0) {
        res++;
        max /= 10;
    }
    return res;
}

// arr[l..r]排序  ,  digit ; L,R 范围上 最大值有digit位
// l..r    3 56 17 100    3（最大值100为3位）
public static void radixSort(int[] arr, int L, int R, int digit) {
    // 以10为基底
    final int radix = 10;
    int i = 0, j = 0;
    // 有多少个数准备多少个辅助空间
    int[] help = new int[R - L + 1];
    // 有多少位就进出几次 比如个位先入桶（d=1） 十位再入桶 百位再入桶 ……
    for (int d = 1; d <= digit; d++) {
        // 固定10个空间
        // count[0] 当前位(d位)是0的数字有多少个
        // count[1] 当前位(d位)是(0和1)的数字有多少个
        // count[2] 当前位(d位)是(0、1和2)的数字有多少个
        // count[i] 当前位(d位)是(0~i)的数字有多少个

        // 先得到count的正确值
        int[] count = new int[radix]; // count[0..9]
        for (i = L; i <= R; i++) {
            // 取出对应的个位/十位...的数字
            // 103  1   3 //i为1 取出的个位为3
            // 209  1   9
            j = getDigit(arr[i], d);
            // 代表某一位进入了count
            count[j]++;
        }

        // 变成count'
        for (i = 1; i < radix; i++) {
            count[i] = count[i] + count[i - 1];
        }

        // 从后遍历arr 插入对应的help数组中
        for (i = R; i >= L; i--) {
            // 
            j = getDigit(arr[i], d);
            help[count[j] - 1] = arr[i];
            count[j]--;
        }

        // 把help中的元素放入arr
        for (i = L, j = 0; i <= R; i++, j++) {
            arr[i] = help[j];
        }
    }
}

// 获取具体个位数（d为1）/十位数 的值为多少
public static int getDigit(int x, int d) {
    return ((x / ((int) Math.pow(10, d - 1))) % 10);
}
```



# 3 排序算法的稳定性

## 概念

- 稳定性是指同样大小的样本再排序之后不会改变相对次序
- 对基础类型来说，稳定性毫无意义
- 对非基础类型来说，稳定性有重要意义
- 有些排序算法可以实现成稳定的，而有些排序算法无论如何都实现不成稳定的

## 排序算法分析

![2](https://dabaixiaozhu.oss-cn-shenzhen.aliyuncs.com/LinBlog/mashibing/basicStructure/class5/stability/2.jpg)

## 排序算法总结

- 1 不基于比较的排序，对样本数据有严格要求，不易改写
- 2 基于比较的排序，只要规定好两个样本怎么比大小就可以直接复用
- 3 基于比较的排序，时间复杂度的极限是O(N*logN)，无法更低了
- 4 时间复杂度O(N*logN)、额外空间复杂度低于O(N)、且稳定的基于比较的排序是不存在的
- 5 为了绝对的速度选快排、为了省空间选堆排、为了稳定性选归并

| 排序算法                 | 时间复杂度 | 额外空间复杂度 | 稳定性 |
| ------------------------ | ---------- | -------------- | ------ |
| 选择排序                 | O(N^2)     | O(1)           | 无     |
| 冒泡排序                 | O(N^2)     | O(1)           | 有     |
| 插入排序                 | O(N^2)     | O(1)           | 有     |
| 归并排序                 | O(N*logN)  | O(N)           | 有     |
| 随机快排（常数时间最少） | O(N*logN)  | O(logN)        | 无     |
| 堆排序                   | O(N*logN)  | O(1)           | 无     |
| 计数排序                 | O(N)       | O(M)           | 有     |
| 基数排序                 | O(N)       | O(N)           | 有     |



# 4 常见的坑

- 1 归并排序的额外空间复杂度可以变成O(1)，“归并排序 内部缓存法”，但是将变得不再稳定；直接用堆

- 2 “原地归并排序" 是垃圾贴，会让时间复杂度变成O(N^2)；直接选插入排序

- 3 快速排序稳定性改进，“01 stable sort”（论文），但是会对样本数据要求更多；直接用桶排序

- 4 在整型数组中，请把奇数放在数组左边，偶数放在数组右边，要求所有奇数之间原始的相对次序不变，所有偶数之间原始相对次序不变（稳定的）；时间复杂度做到O(N)，额外空间复杂度做到O(1)
